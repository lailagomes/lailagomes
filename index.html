Como Publicar Este Currículo no GitHub

1

Criar Conta no GitHub

Acesse github.com e crie uma conta gratuita se ainda não tiver.

2

Criar Novo Repositório

Após login, clique em "+" no canto superior direito e selecione "New repository". Dê o nome: seu-usuario.github.io (substitua "seu-usuario" pelo seu nome de usuário).

3

Adicionar Arquivos

Na página do repositório, clique em "Add file" > "Create new file". Nomeie o arquivo como index.html e cole todo o código desta página no campo de texto.

4

Publicar o Site

Role até o final da página, clique em "Commit new file". Seu site estará disponível em: https://seu-usuario.github.io em alguns minutos.
